# AlphaTerminal

A real-time financial disclosure dashboard integrated with AI analysis.

## Project Structure

*   **/frontend**: Next.js application (Dashboard UI)
*   **/backend**: FastAPI application (Scraping & AI Analysis)

## Development Setup

### fast start

1.  **Frontend**
    ```bash
    cd frontend
    npm install
    npm run dev
    ```
    Access: http://localhost:3000

2.  **Backend**
    ```bash
    cd backend
    pip install -r requirements.txt
    python main.py
    ```
    Access: http://localhost:8000/docs
